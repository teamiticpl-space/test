# จำลองพยากรณ์อากาศ + ความเสี่ยงการเพาะปลูกประเทศไทย

> Thai Weather Forecast & Crop-Risk Simulator

เว็บแอปพยากรณ์อากาศ 15 วันครบทั้ง 77 จังหวัด พร้อมประเมิน **ความเสี่ยงต่อผลผลิตพืชไร่ตามวันปลูก/หว่าน**
แนะนำพันธุ์ ปุ๋ยรายระยะตามชนิดดิน และเฝ้าระวังโรค/แมลงตามสภาพอากาศ — ออกแบบให้เกษตรกร/ผู้ใช้ทั่วไปใช้งานง่าย

## ✨ ฟีเจอร์หลัก

- **แผนที่ความเสี่ยงรายจังหวัด** — หมุด 77 จังหวัด สีเขียว/เหลือง/แดงตามระดับความเสี่ยง คลิกเลือกจังหวัดได้
- **พยากรณ์ 15 วัน** — อุณหภูมิสูง/ต่ำ ปริมาณฝน ความเร็วลม ความชื้น + กราฟแนวโน้มรายวัน
- **จำลองสถานการณ์ (what-if)** — ปรับปริมาณฝน % / อุณหภูมิ °C / ความเร็วลม % แล้วผลกระทบสะท้อนทันที
- **คะแนนความเสี่ยง 0-100** พร้อมหน้าปัดและองค์ประกอบรายปัจจัย
- **โหมดพืช 4 ชนิด** + โหมดสภาพอากาศทั่วไป

### พืชที่รองรับ

| พืช | อายุ | จุดวิกฤต | โรค/แมลงเด่น |
|---|---|---|---|
| ข้าว | 95–145 วัน | ร้อน/เย็นช่วงออกดอก, ฝน/ลมช่วงเก็บเกี่ยว | โรคไหม้คอรวง, เพลี้ยกระโดด |
| อ้อย | ~11–12 เดือน | ขาดน้ำช่วงย่างปล้อง, น้ำขังช่วงสะสมน้ำตาล | หนอนกออ้อย, เหี่ยวเน่าแดง |
| มันสำปะหลัง | ~8–12 เดือน | น้ำขังช่วงลงหัว (หัวเน่า) | เพลี้ยแป้ง, ไรแดง, ใบด่าง |
| ข้าวโพดเลี้ยงสัตว์ | ~100–120 วัน | ขาดน้ำ+ร้อนช่วงออกไหม | หนอนกระทู้ลายจุด (FAW) |

แต่ละพืชเลือกพันธุ์ (อายุ/ความทนต่างกัน) และชนิดดิน (เหนียว/ร่วน/ทราย) ได้
ระบบจะแนะนำแผนปุ๋ยรายระยะและเฝ้าระวังโรค/แมลงตามสภาพอากาศที่จำลอง

## 🧮 วิธีคิดคะแนนความเสี่ยงพืช

ระบบนับอายุพืชจากวันปลูกตามพันธุ์ แล้วจับคู่ช่วงพยากรณ์ 15 วันกับ **ระยะการเจริญเติบโต**
แต่ละระยะอ่อนไหวต่อความเครียด 5 อย่าง (ฝนหนัก/ฝนทิ้งช่วง/ร้อน/ลม/เย็น) ไม่เท่ากัน
คะแนนรวม = ปัจจัยที่รุนแรงที่สุดในวันที่เสี่ยงที่สุด (ประเมินเฉพาะระยะที่อยู่ในหน้าต่างพยากรณ์ 15 วัน)

> ⚠️ คำแนะนำปุ๋ย/โรค-แมลงเป็นแนวทางทั่วไปเพื่อการวางแผน อัตราจริงควรยึดตามฉลากผลิตภัณฑ์
> ผลวิเคราะห์ดิน และคำแนะนำของเกษตรอำเภอ/กรมวิชาการเกษตร เครื่องมือนี้ไม่ใช่คำเตือนภัยอย่างเป็นทางการ

## 🛠 เทคโนโลยี

- **Next.js 14** (App Router) + **TypeScript** (strict)
- **Tailwind CSS** + คอมโพเนนต์สไตล์ shadcn/radix
- **Zustand** (state), **Recharts** (กราฟ), **react-leaflet** + OpenStreetMap (แผนที่), **Framer Motion** (แอนิเมชัน)
- ข้อมูลพยากรณ์จาก **[Open-Meteo](https://open-meteo.com/)** (ฟรี ไม่ต้องใช้ API key)
- ฟอนต์ Noto Sans Thai

หากเชื่อมต่อ Open-Meteo ไม่ได้ ระบบจะใช้ข้อมูลจำลองตามฤดูกาลไทยโดยอัตโนมัติ (มีป้ายบอกแหล่งข้อมูล)

## 🚀 เริ่มใช้งาน

```bash
npm install
npm run dev
# เปิด http://localhost:3000
```

build สำหรับ production:

```bash
npm run build
npm run start
```

## 📁 โครงสร้างหลัก

```
app/                      # Next.js App Router (layout, page, loading)
components/               # UI: dashboard, แผนที่, การ์ดพยากรณ์, กราฟ, gauge, ตัวเลือกพืช ฯลฯ
lib/
  provinces.ts            # 77 จังหวัด + พิกัด + ภูมิภาค
  weather-api.ts          # ดึง Open-Meteo (batch ทุกจังหวัดในคำขอเดียว)
  mock-weather.ts         # ข้อมูลจำลองตามฤดูกาล (fallback)
  scenario.ts             # ปรับค่าจำลองสถานการณ์
  risk/
    risk-engine.ts        # ความเสี่ยงสภาพอากาศ (ถ่วงน้ำหนัก)
    crops/                # เครื่องยนต์ความเสี่ยงพืชแบบ generic + ข้อมูลแต่ละพืช
      types.ts            #   โมเดล/การให้คะแนนกลาง
      rice.ts sugarcane.ts cassava.ts maize.ts
      index.ts            #   ทะเบียนพืช
stores/weather-store.ts   # Zustand store
```

เพิ่มพืชใหม่ = เพิ่มไฟล์ข้อมูลใน `lib/risk/crops/` แล้วลงทะเบียนใน `index.ts`
